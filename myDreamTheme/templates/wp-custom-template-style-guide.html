Images and Galleries

Note: You will need to upload your own (temporary) images to see the changes happening in real time. Images are not part of themes and are part of advanced coding features -- they can be included in a theme's "assets" folder, but if you leave an uploaded image in your theme, the file path will be back to the Local Environment where you planned your site. For this course, the recommendation is you forego using images in the finished product; use test or dummy content, but make sure to remove it when you are finished.
